# Portafolio Personal — cv-astro

Sitio web de portafolio personal construido con Astro, React y Tailwind CSS.

## 📂 Estructura del proyecto

```text
/
├── public/
│   ├── files/
│   └── image/
├── src/
│   ├── components/
│   ├── content/
│   ├── layouts/
│   ├── pages/
│   └── styles/
└── package.json
```

## 🚀 Tecnologías

| Tecnología | Versión | Uso |
| :--------- | :------ | :-- |
| [Astro](https://astro.build) | ^6.1.5 | Framework principal (SSG/SSR) |
| [React](https://react.dev) | ^19.2.5 | Componentes interactivos |
| [Tailwind CSS](https://tailwindcss.com) | ^4.2.2 | Estilos y diseño |
| [TypeScript](https://www.typescriptlang.org) | — | Tipado estático |
| [Iconify](https://iconify.design) | 1.0.7 | Iconos |

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto:

| Comando | Acción |
| :------ | :----- |
| `npm install` | Instala las dependencias |
| `npm run dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `npm run build` | Genera el sitio de producción en `./dist/` |
| `npm run preview` | Previsualiza el build antes de desplegar |
| `npm run astro ...` | Ejecuta comandos de la CLI de Astro |
| `npm run astro -- --help` | Muestra la ayuda de la CLI de Astro |
